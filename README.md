## 💻기능

- 회원가입
- 로그인
- 게시글 인피니트 스크롤
- 게시판 등록, 수정, 삭제
- 댓글 등록
- 게시글 태그 검색
- 카테고리 태그 검색
- 비밀번호 변경
<br/><br/>

- React, JavaScript ES6, SCSS, HTML
- Express, NodeJS
- Mongo DB
- Redux & Redux-saga
- JWT Web Token
- Ant Design, Reactstrap
- Infinite Scroll(Intersection Observer)
- CKEditor5
